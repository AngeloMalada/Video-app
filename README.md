Tech stack

- NextJs
- React
- Typescript
- Sanity
- TailwindCss
- Zustand
- React router

Installation

Clone the repo

```

git clone https://github.com/AngeloMalada/Video-app

```

Install NPM packages (also install them in video-backend)

```

npm install

```

Start the project (also in video-backend for sanity start it with sanity start)

```

npm run dev

```
